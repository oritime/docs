# 起源云文档中心

欢迎来到 **起源云（OriginCloud）帮助文档**。  
这里汇总了从**注册账号 → 创建实例 → 上传数据 → 启动训练 → 保存镜像**的完整流程。

> 起源云是新一代智算云平台，提供无卡创建、秒级计费、AI终端、MCP原生支持等功能，  
> 让 AI 训练像打开笔记本一样简单。
