## 起源云是什么？能做什么？

**起源云是一个提供 GPU 算力的云平台。**

简单来说，就是你可以租用一台带 GPU 的远程电脑，在上面运行 AI 训练、推理等任务。

**主要用途包括：**
- 训练 AI 模型（比如训练 ChatGPT 这样的大语言模型）
- 运行 AI 推理（比如用训练好的模型生成图片、文本）
- 处理大量数据（需要 GPU 加速的计算任务）
- 学习和实验（想试试 GPU 编程，但自己没买显卡）

**不需要自己买昂贵的 GPU 显卡**，按需租用，用多少付多少。

---

## 什么是实例？

**实例就是你在起源云上租用的那台"电脑"。**

你可以把它理解成：
- 一台远程的 Linux 服务器
- 有 CPU、内存、硬盘
- 最重要的是有 GPU（显卡）
- 可以安装软件、运行代码、存储数据

**特点：**
- 随时可以创建、关机、删除
- 关机后停止计费，但数据会保留
- 可以随时调整配置（比如换更强的 GPU）

详细说明：[快速上手](/getting-started/quickstart)

## 什么是 GPU？为什么需要 GPU？

**GPU 就是显卡，但这里不是用来打游戏的，是用来做计算的。**

**为什么需要 GPU？**
- CPU 适合做复杂但串行的任务（一次做一件事）
- GPU 适合做简单但并行的任务（同时做很多件事）
- AI 训练需要同时处理大量数据，GPU 比 CPU 快几十倍甚至上百倍

**举个例子：**
- 用 CPU 训练一个模型可能需要几天
- 用 GPU 可能只需要几小时

**起源云提供各种型号的 GPU：**
- 高端：A100、H100（适合训练大模型）
- 中端：RTX 4090、A6000（适合大多数任务）
- 入门：RTX 3090 等（适合学习和实验）

不知道选哪个？看这里：[如何选择 GPU](/getting-started/choose-gpu)

## 什么是镜像？

**镜像就是预装好系统和软件的"模板"。**

就像装电脑系统一样：
- 普通镜像：只装了操作系统（Linux）
- 官方镜像：已经装好了 PyTorch、TensorFlow 等常用框架，开箱即用

**三种镜像：**
- **官方镜像**：起源云提供的，已经装好常用软件，推荐新手使用
- **自定义镜像**：你自己保存的环境，下次创建实例时可以直接用
- **共享镜像**：其他用户分享的，可以直接使用

**选择建议：**
- 第一次用：选官方镜像，省事
- 有特殊需求：自己装好软件后保存为镜像，下次直接用

详细说明：[镜像管理](/image-mgmt/overview)

## 怎么连接和使用实例？

**有三种方式，都很简单：**

### 1. JupyterLab（推荐新手）

在浏览器里打开，就像用网页版的 Python 编辑器。

- 不需要安装任何软件
- 可以直接写代码、运行、上传文件
- 界面友好，适合新手

### 2. SSH（适合有经验）

用命令行连接，就像远程登录服务器。

- 适合熟悉 Linux 的用户
- 可以后台运行长时间任务

### 3. VNC（需要图形界面）

可以看到完整的桌面环境。

- 适合需要图形界面的应用
- 在浏览器里打开，不需要装客户端

详细说明：[连接实例](/instances/connect)

## 数据存在哪里？会丢失吗？

**起源云有几种存储空间：**

### 系统盘（30GB）
- 存操作系统、安装的软件、代码
- **注意**：删除实例后会丢失（除非保存为镜像）

### 数据盘（可扩容）
- 存数据集、模型等大文件
- **注意**：删除实例后会丢失

### 起源云盘（推荐）
- **不限容量**，所有实例都能访问
- **永久保存**，删除实例后数据还在
- 可以在网页上传、下载文件

**建议：**
- 重要数据存在起源云盘，这样删除实例也不怕丢数据
- 临时数据可以存在数据盘

详细说明：[存储系统](/storage/overview)

## 怎么收费？贵不贵？

**两种计费方式：**

### 按量计费（推荐新手）
- 用多少付多少，按秒计费
- 关机后立即停止计费
- 灵活，适合不确定用多久的情况

### 包日/包周/包月
- 预付费，价格更优惠
- 即使关机也会继续计费，直到到期
- 适合确定要用很长时间的情况

**省钱技巧：**
- 用完及时关机（按量计费）
- 设置定时关机，避免忘记关机
- 重要数据存起源云盘，可以放心删除实例

详细说明：[计费说明](/getting-started/pricing)

## 需要什么技术基础吗？

这个问题其实取决于你想用起源云做什么。别担心，无论你的基础如何，都有适合你的使用方式。

如果你只是想体验一下 AI 模型，或者运行一些现成的代码，那门槛其实很低。
- 你不需要精通编程，甚至不需要会写代码。
- 起源云提供了 JupyterLab 这样的网页界面，你只需要跟着教程，点点鼠标，就能运行一些有趣的 AI 应用。比如生成图片、文本对话这些，都可以在几分钟内上手。

当然，如果你想深入一些，比如训练自己的模型、做自己的 AI 项目，那确实需要一些技术基础。
- 你需要会 Python 编程，简单的 Linux 系统的操作，了解一些深度学习的基础知识，熟悉 PyTorch 或 TensorFlow 这样的框架。
- 但即使是这样，也不用太担心。很多用户都是从零开始学习的，关键是要有耐心和好奇心。

更重要的是，起源云在设计时就考虑到了新手的需求。
- 我们提供了详细的文档和教程，每一步都有清晰的说明。环境都是预装好的，你不用自己配置那些复杂的依赖和库。界面也尽量做得友好，让你能专注于学习和实验，而不是被技术细节困扰。

所以，如果你是完全的新手，不妨先从简单的开始吧！比如用 JupyterLab 运行一些示例代码，感受一下 GPU 的威力。如果你已经有了一些基础，那可以直接开始你的项目。起源云会一直陪伴你成长和进步的♥
